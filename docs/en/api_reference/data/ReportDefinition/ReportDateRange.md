

# ReportDateRange

The ReportDateRange object serves the report compilation target period.

### Service

+ [ReportDefinitionService](../../services/ReportDefinitionService.md)

### Namespace

[ReportDefinitionService#Namespace](../../services/ReportDefinitionService.md#namespace)

| Field | Type | Description | response |
| ----- | ---- | ----------- | -------- |
| startDate | xsd:string | The compilation start date. This item is required if DateRangeType is set to &#34;CUSTOM DATE&#34;. | yes | |
| endDate | xsd:string | The compilation end date. This item is required if DateRangeType is set to &#34;CUSTOM DATE&#34;. | yes | |
