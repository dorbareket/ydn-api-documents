

# ThirdPartyTrackingDomain

ThirdPartyTrackingDomain object indicates the domain that can be used for the src of third party tracking scripting tag.

### Service

+ [DictionaryService](../../services/DictionaryService.md)

### Namespace

[DictionaryService#Namespace](../../services/DictionaryService.md#namespace)

| Field | Type | Description | response |
| ----- | ---- | ----------- | -------- |
| domain | xsd:string | Domain | yes | |
| vendor | xsd:string | Third Party Tracking vendor | yes | |
| vendorName | xsd:string | Logical name of Third Party Tracking vendor | yes | |
