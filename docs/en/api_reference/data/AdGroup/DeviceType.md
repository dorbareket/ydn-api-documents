

# DeviceType (enum)

DeviceType displays types of devices to be posted on.

#### Service

+ [AdGroupService](../../services/AdGroupService.md)

#### Namespace

[AdGroupService#Namespace](../../services/AdGroupService.md#namespace)

| Enumeration  |       Type       |          Description          |
| ------------ | ---------------- | ----------------------------- |
| DESKTOP | xsd:string | PC |
| WAP_MOBILE | xsd:string | Contemporary mobile phones (Feature Phone) |
| SMARTPHONE | xsd:string | Smartphones |
| TABLET | xsd:string | Tablet device |
| NONE | xsd:string | Deliver to all usable devices.<br/>*Setting NONE is only available for AdGroupService. |
