# ErrorDetail
The ErrorDetail object serves error content details.
### Service
Same for all services:

| Field | Data Type | Description | 
|---|---|---|
| requestKey| xsd: string| The field in which an error occurred. |
| requestValue| xsd: string[]| The value for which an error occurred. |
