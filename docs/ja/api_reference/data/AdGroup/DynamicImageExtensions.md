# DynamicImageExtensions (enum)
DynamicImageExtensionsは、画像自動付与の設定を表します。
### Service
+ [AdGroupService](../../services/AdGroupService.md)

### Namespace
[AdGroupService#Namespace](../../services/AdGroupService.md#namespace)

| 値 | 説明 |
|---|---|
| ACTIVE| 画像自動付与を設定します。 |
| PAUSED| 画像自動付与を設定しません。 |
