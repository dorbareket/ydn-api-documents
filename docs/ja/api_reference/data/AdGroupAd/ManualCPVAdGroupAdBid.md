

# ManualCPVAdGroupAdBid

ManualCPVAdGroupAdBidオブジェクトは、広告の手動入札方法（CPV）を表します。
※広告掲載方式がVIDEO_AD(動画広告）のキャンペーンIDを指定した時のみ利用できます。
