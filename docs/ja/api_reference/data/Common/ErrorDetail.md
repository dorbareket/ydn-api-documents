# ErrorDetail
ErrorDetailオブジェクトは、エラーの詳細な内容を表します。
### Service
全サービス共通

| フィールド | データ型 | 説明 | 
|---|---|---|
| requestKey| xsd: string| エラーが発生したフィールドです。 |
| requestValue| xsd: string[]| エラーが発生した値です。 |
