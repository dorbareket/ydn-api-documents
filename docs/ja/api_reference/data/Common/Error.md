# Error
Errorオブジェクトは、エラー内容を表します。
### Service
全サービス共通

| フィールド | データ型 | 説明 |
|---|---|---|
| code| xsd: string| エラーコードです。 |
| message| xsd: string| エラーメッセージです。 |
| detail[]| <a href="ErrorDetail.md">ErrorDetail</a>| エラー内容の詳細です。 |
