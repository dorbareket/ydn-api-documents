

# UploadUrlValue

UploadUrlValueオブジェクトは、アップロード先URL情報を表します。

### Service

+ [VideoService](../../services/VideoService.md)

### Namespace

[VideoService#Namespace](../../services/VideoService.md#namespace)

| Field | Type | Description | response | get | set | remove |
| ----- | ---- | ----------- | -------- | --------- | --------- | --------- |
| accountId | xsd:long | アカウントIDです。 | yes | - | - | - | |
| acceptableUploadStatus | xsd:boolean | trueの場合、アップロードが可能です。 | yes | - | - | - | |
| uploadUrl | xsd:string | アップロード先URLです。 | yes | - | - | - | |
