# RuleCondition
RuleConditionオブジェクトは、ターゲットリストに設定可能な条件を表します。

### Service

| compareOperator| enum <a href="./CompareOperator.md">CompareOperator</a>| Requirement | Requirement | Ignore | ルールに設定するオペレーター。 |
